About me

發佈於 2020-01-06

Hey there! I’m Easycker (Alan Wang), a software engineer living on the Internet.

我的 blog 在互聯網上的各個平臺兜兜轉轉了这么些年，终于在 2019 年的最後一天回到了 Github 上。
這是我在比特之海中分享自己所見、所得、所感的一小塊自留地，感謝你的訪問。

本站尚不開放評論功能，如果你有想法與我進行交流，歡迎使用電郵。

有關本站的一些技術背景：

使用靜態網頁生成器 Hexo 生成。
使用 Makito 創作的 Hexo 主題 Typography，感謝他創作出這麼漂亮的版式。
託管於 GitHub Pages。
主要的文章和代碼使用 VSCode 編寫。
最後一定要感謝發明了 Git 和 Markdown 的兩位前輩，Linus Torvalds & John Gruber.

Have a nice day.
